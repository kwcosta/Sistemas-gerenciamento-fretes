# Sistema de Gerenciamento de Fretes

## 📖 Descrição
Este projeto foi desenvolvido como um exercício acadêmico para aplicar conceitos de lógica de programação. Trata-se de um sistema simples para gerenciar notas fiscais de fretes, com o objetivo de extrair e organizar dados para facilitar a análise de valores e padrões.

## ✨ Funcionalidades
- Cadastro e leitura de informações de notas fiscais.
- Preparação das informações para análise em planilhas, permitindo a comparação de valores.

## 🛠️ Tecnologia Utilizada
- **Portugol**

## 🚀 Como Executar
Para executar este projeto, é necessário ter o [Portugol Webstudio que você usa para rodar Portugol] instalado.

1. Abra o programa ou entre pelo navegador
2. selecione o código padrão do programa, apague e cole o código do Gerenciamento de fretes e execute o código
